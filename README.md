# No Goal Timer

No Goal Timer removes the post-score countdown in Super Battle Golf.

After a player scores, the round no longer starts a forced timer. The hole waits until the remaining active players finish.

## Features

- Disables post-score countdown trigger
- No configuration required
- Works with BepInEx on the Mono build

## Installation

Install through Thunderstore Mod Manager, or manually place `NoGoalTimer.dll` inside your BepInEx plugins folder.

## Compatibility

- Game: Super Battle Golf
- Runtime: BepInEx 5 (Mono)

## Notes

This mod patches the game method that starts the match-end countdown after a player scores.
